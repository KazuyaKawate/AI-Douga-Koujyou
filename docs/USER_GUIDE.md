# ユーザーガイド — AI動画工場 v3.1

## 初回セットアップ

### 1. 環境変数

`.env` ファイルを作成し、OpenAI APIキーを設定します。

```
OPENAI_API_KEY=sk-...
APP_ENV=development
```

### 2. スタジオ設定を確認

⚙️ **スタジオ設定** を開き、以下を確認・設定してください。

- **AI設定**: モデル（`gpt-4o-mini` 推奨）、コスト節約モード、開発モード
- **生成設定**: 画像/動画/音声プロバイダー（デフォルト: Manual）
- **プロジェクト設定**: デフォルトエピソード長・シーン数

## 基本ワークフロー

### Step 1: キャラクターを設定する

🧑 **キャラクター管理** ページを開きます。

**新規作成:**
1. 左サイドバーの「新しいキャラクターを追加」で名前と役割を入力
2. 「✚ 作成」ボタンで作成
3. 5つのタブで詳細を設定:
   - 📝 基本情報: 年齢・性別・役割
   - 👤 外見: 髪型・服装・アクセサリー
   - 🎭 性格: 話し方・一人称・キャッチフレーズ
   - 🎨 プロンプト: 画像/動画/音声プロンプト基盤
   - 📁 アセット: ポートレート画像・音声サンプル
4. 「💾 変更を保存」でJSON保存
5. サイドバーの「デフォルトキャラクター」に設定

### Step 2: 背景を設定する

🏞️ **背景管理** ページを開きます。

**新規作成:**
1. 「新しい背景を追加」で名前とカテゴリを入力
2. 5つのタブで設定:
   - 📝 基本情報: カテゴリ・説明
   - 🏙️ ビジュアル: ロケーション・時間帯・照明・雰囲気
   - 🎥 カメラ: デフォルトアングル・レンズ・モーション
   - 🎨 プロンプト: 画像/動画プロンプト基盤・ネガティブ
   - 📁 アセット: 参考画像・レイアウト
3. 保存してデフォルト背景に設定

### Step 3: プロンプトテンプレートを作成する

📝 **プロンプトビルダー** ページを開きます。

1. キャラクター・背景・出力タイプ・ムード・スタイル・カメラを選択
2. 「🔧 プロンプトを生成」をクリック
3. 生成されたプロンプトを確認
4. 「💾 テンプレートとして保存」で名前を付けて保存

**演出ディレクションの注入（AI Director連携）:**
- サイドバーの「🎬 演出ディレクション」でエピソードとシーンを選択
- 「🎬 演出を適用」でシーン方向性をロード
- 「✏️ プロンプトに追記」でプロンプトテキストに追記

### Step 4: エピソードを生成する

⚡ **一発生成** ページを開きます。

1. **テーマ**: 動画のテーマ・タイトルを入力
2. **ターゲット**: 視聴者層（任意）
3. **エピソードID**: `EP01` のような識別子
4. **動画の長さ**: 1〜5分
5. **スタイル**: 解説・教育 / エンタメ / ニュース等
6. **シーン数**: 2〜8（コスト節約時は2〜3推奨）
7. サイドバーでキャラクター・背景・テンプレートを選択
8. 「⚡ 一発生成スタート」をクリック

生成後、以下のファイルが `project/EPXX/` に保存されます:
- `episode.json` — エピソードデータ
- `EPXX_image_prompts.txt` — 画像プロンプト（シーン別）
- `EPXX_video_prompts.txt` — 動画プロンプト（シーン別）
- `EPXX_voice_script.txt` — 音声台本
- `EPXX.srt` — 字幕

### Step 5: 演出を設計する (AI Director)

🎭 **AI Director** ページを開きます。

**手動モード:**
1. エピソードを選択
2. 全体演出設定（トーン・スタイル・ペーシング）を入力
3. 各シーンのエキスパンダーを開き、カメラ・感情・プロンプト演出ノートを入力
4. 「💾 演出計画を保存」

**AI生成モード:**
1. 右カラムの「⚙️ AI生成設定」を展開
2. 全体トーン・感情・スタイル・ペーシングを指定
3. キャラクターと背景を参考として選択（任意）
4. 「🤖 AI演出計画を生成」をクリック（10〜20秒）
5. 生成結果を確認・編集し「💾 演出計画を保存」

### Step 6: 画像・動画・音声を手動生成する

生成されたプロンプトを各ツールでご利用ください:

| 素材 | 推奨ツール | プロンプトファイル |
|------|-----------|-----------------|
| 画像 | Nano Banana / Midjourney / DALL-E 3 | `EPXX_image_prompts.txt` |
| 動画 | Runway Gen-3 Alpha / Kling AI | `EPXX_video_prompts.txt` |
| 音声 | Nano Banana / VOICEVOX / ElevenLabs | `EPXX_voice_script.txt` |
| BGM | Suno AI / Udio | `director_plan.json` の bgm_direction |

生成したファイルは以下に配置してください:
- 画像: `project/EPXX/assets/images/`
- 動画: `project/EPXX/videos/`
- 音声: `project/EPXX/voices/`

### Step 7: 制作管理でチェックリストを更新する

🎬 **制作管理** ページを開きます。

1. エピソードを選択
2. 「🔍 自動検出して更新」でファイルを検出
3. 各ステージを手動で「✅ 完了」にマーク:
   - 📝 台本・スクリプト
   - 🖼️ 画像制作
   - 🎬 動画クリップ
   - 🎙️ 音声ナレーション
   - 🎵 BGM（任意）
   - 🔊 効果音（任意）
4. 制作ノートを記録
5. 「📦 パッケージを書き出す」でエクスポート

### Step 8: 動画を組立てる

✂️ **動画組立** ページで FFmpeg を使って最終動画を生成します。

## ダッシュボードの見方

📊 **制作ダッシュボード** でエピソード一覧を確認できます。

**バッジ:**
- `📦 書き出し済` — エクスポートパッケージあり
- `🎬 制作可能` — 主要ステージ完了
- `🎬 Director` — 演出計画あり

**ステータス:**
- `✅ 完成` — 最終動画ファイルあり
- `🔄 制作中` — 素材収集中
- `⏳ 素材待ち` — プロンプトあり、素材未収集
- `⏸ 未着手` — episode.json のみ

## よくある質問

**Q: OpenAI APIキーはどこで取得できますか？**  
A: https://platform.openai.com/api-keys からAPIキーを発行してください。

**Q: コスト節約モードとは何ですか？**  
A: OpenAIへのプロンプトを短縮してトークン使用量を削減します。2〜3シーン程度なら同等の品質が得られます。

**Q: 生成した project/ ファイルをGitにコミットしますか？**  
A: しません。`.gitignore` で除外されています。ローカルでのみ管理されます。

**Q: AI Director の演出計画は必須ですか？**  
A: 任意です。演出計画なしでも一発生成・制作管理は問題なく動作します。

**Q: 複数エピソードを管理できますか？**  
A: はい。`EP01`, `EP02`, ... と異なるIDで生成することで複数エピソードを管理できます。
