# AI動画工場

AIツールを組み合わせて、スクリプト生成から動画完成までを自動化・効率化する動画制作プロジェクト。

---

## プロジェクトの目的

ChatGPT・Claude Code・Runway・Nano Banana・Canvaそれぞれの強みを活かし、
アイデアから完成動画までのワークフローを標準化・再現可能にすること。

---

## フォルダ構成と役割

| フォルダ | 役割 |
|---|---|
| `scripts/` | 動画の台本・ナレーション原稿 |
| `prompts/` | 各AIツールへの入力プロンプトテンプレート |
| `images/` | 生成・収集した画像素材 |
| `voices/` | 生成した音声ファイル（ナレーション等） |
| `videos/` | 生成・収集した動画クリップ |
| `subtitles/` | 字幕ファイル（SRT・VTT形式等） |
| `output/` | 完成した最終動画 |
| `config/` | 設定ファイル・APIキー定義（テンプレート） |
| `assets/` | BGM・ロゴ・フォント等の共通素材 |
| `temp/` | 作業中の一時ファイル（Gitには含まない） |

---

## 動画制作フロー

```
1. 企画・台本生成
   └─ ChatGPT でテーマ・構成・ナレーション原稿を生成
      └─ scripts/ に保存

2. 画像・動画生成
   └─ Runway で映像クリップを生成
      └─ videos/ に保存
   └─ 必要に応じて画像を images/ に保存

3. ナレーション音声生成
   └─ Nano Banana でテキストから音声を生成
      └─ voices/ に保存

4. 字幕生成・整形
   └─ Claude Code で字幕ファイル（SRT等）を自動生成・整形
      └─ subtitles/ に保存

5. 編集・仕上げ
   └─ Canva で映像・音声・字幕を組み合わせて最終動画を編集
      └─ output/ に書き出し

6. 公開・配信
   └─ YouTube・SNS等に投稿
```

---

## 各ツールの役割

### ChatGPT
- テーマ発案・リサーチ
- 台本・ナレーション原稿の生成
- プロンプト設計のサポート
- タイトル・説明文・ハッシュタグの生成

### Claude Code
- 字幕ファイル（SRT・VTT）の自動生成・整形
- スクリプトやプロンプトのファイル管理・自動化
- 繰り返し作業のスクリプト化（バッチ処理等）
- プロジェクト構成の維持・更新

### Runway
- テキストや画像から動画クリップを生成（Text-to-Video / Image-to-Video）
- 映像の編集・エフェクト適用
- BGロール映像・トランジション素材の生成

### Nano Banana
- ナレーションテキストから自然な音声を生成
- 複数の話者スタイル・感情表現に対応
- 生成音声を voices/ に書き出し

### Canva
- 動画クリップ・音声・字幕の最終編集・組み合わせ
- テンプレートを使ったタイトルカード・エンドカードの作成
- サムネイル画像の作成
- 完成動画の書き出しと共有

---

## 注意事項

- `config/` 内のAPIキーや認証情報は `.gitignore` で除外してください。
- `temp/` フォルダは作業中の一時ファイル置き場です。Gitには含まれません。
- 完成した素材は必ず対応するフォルダに整理してから次のステップへ進んでください。
