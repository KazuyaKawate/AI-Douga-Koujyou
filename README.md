# 🎬 AI動画工場 v3.1 RC1

AI動画工場は、AIを活用したローカル動画制作自動化プラットフォームです。
テーマを入力するだけで台本・シーン分割・画像/動画プロンプト・字幕・音声台本を全自動生成し、手動制作フローをガイドします。

> **First Public Release Candidate — 外部生成API（画像・動画・音声）は現在手動モードのみ対応。**  
> AIテキスト生成（OpenAI）のみ自動で呼び出します。

---

## クイックスタート

| ステップ | 操作 |
|---------|------|
| **Step 1** | デスクトップの `AI動画工場.lnk` をダブルクリック |
| **Step 2** | ターミナルが開きサーバーが起動するまで待つ（初回は 10〜20 秒） |
| **Step 3** | ブラウザが自動で `http://localhost:8501` を開く |

> ショートカットがない場合は `create_desktop_shortcut.ps1` を右クリック → **PowerShell で実行** してください。  
> 環境を確認するには `check_environment.bat` を実行してください。  
> 全ユーティリティの詳細は [README_UTILITIES.md](README_UTILITIES.md) を参照してください。

## セットアップ

### 1. リポジトリをクローン

```bash
git clone https://github.com/KazuyaKawate/AI-Douga-Koujyou.git
cd AI-Douga-Koujyou
```

### 2. 依存パッケージをインストール

```bash
pip install -r requirements.txt
```

### 3. 環境変数を設定

`.env.example` をコピーして `.env` を作成し、APIキーを設定します。

```bash
cp .env.example .env
# .env を編集して OPENAI_API_KEY を設定
```

### 4. プロジェクト整合性チェック（任意）

```bash
python scripts/check_project.py
```

### 5. アプリを起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が開きます。

## 使い方ガイド

### エピソードを作成する

1. ⚙️ **スタジオ設定** でプロジェクト名・AIモデルを確認
2. ⚡ **一発生成** でテーマを入力し「⚡ 一発生成スタート」をクリック
3. 生成完了後、プロンプトをコピーして Nano Banana / Runway で画像・動画を手動生成

### キャラクター管理

1. 🧑 **キャラクター管理** でキャラクターを作成（名前・外見・性格・プロンプト）
2. 「デフォルトキャラクター」に設定すると一発生成時に自動適用
3. 画像プロンプトに `【キャラクター設定】` ブロックとして注入される

### 背景管理

1. 🏞️ **背景管理** でロケーション・カメラ・雰囲気を登録
2. デフォルト背景に設定すると一発生成時に自動適用

### プロンプトビルダー

1. 📝 **プロンプトビルダー** でキャラクター・背景・ムード・スタイル・カメラを選択
2. 「🔧 プロンプトを生成」で4種類の出力（画像/動画/ボイスディレクション/サムネイル）を生成
3. 「💾 テンプレートとして保存」でよく使う設定を再利用
4. 🎬 演出ディレクション セクションで AI Director の演出をプロンプトに注入可能

### AI Director で演出を設計する

1. 🎭 **AI Director** でエピソードを選択
2. **手動モード**: 全体トーン・ペーシング・各シーンの演出を手入力
3. **AI生成モード**: 「🤖 AI演出計画を生成」で OpenAI がシーンごとの演出計画を自動生成
   - カメラアングル・モーション・フレーミング・ライティング
   - 画像/動画/音声プロンプト演出ノート
4. 「💾 演出計画を保存」で `project/EPXX/director_plan.json` に保存

### 制作パッケージを書き出す

1. 🎬 **制作管理** でエピソードを選択
2. ファイル検証セクションで必要ファイルの状態を確認
3. 制作チェックリストの各ステージを「✅」でマーク
4. 「📦 パッケージを書き出す」でファイルを `project/EPXX/export/` にまとめる
5. `production_report.json` に制作サマリーが生成される（演出計画情報を含む）

## 開発モード / コスト節約モード

| モード | 説明 | 設定場所 |
|--------|------|----------|
| 開発モード | 外部生成API自動呼び出しを無効化。プロンプトのみ生成 | ⚙️ スタジオ設定 → AI設定 |
| コスト節約モード | OpenAIプロンプトを短縮してトークン消費を削減 | ⚡ 一発生成フォーム内 |

---

## ⚠️ 現在の制限事項

- **画像・動画・音声の自動生成は未対応**です。各プロンプトをコピーして Nano Banana / Runway / VOICEVOX 等で手動生成してください。
- OpenAI API キー（GPT-4o または GPT-4o-mini）が必要です。
- 生成ファイルは `project/` フォルダに保存されます（`.gitignore` で除外済み）。
